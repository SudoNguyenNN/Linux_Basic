## 2. Các loại filesystem phổ biến trên linux

Các loại filesystem được linux hỗ trợ:
  
  - Filesystem cơ bản: EXT2, EXT3, EXT4, XFS, Btrfs,JFS, NTFS,..
  - Filesystem dành cho dạng lưu trữ Flash: thẻ nhớ,..
